# Android Programs

This repository contains a collection of Android applications developed using **Android Studio**.
The programs demonstrate basic Android development concepts such as activities, fragments, menus, forms, and UI components.

## Technologies Used

* Java
* XML
* Android Studio
* Android SDK

## Programs Included

* Menu Implementation
* Fragment Example
* Login Form
* Registration Form
* Resume Layout
* ListView Example
* Basic UI Programs

## Project Structure

Each folder contains the source code of a specific Android program including:

* Java source files
* XML layout files
* AndroidManifest.xml

## Purpose

This repository was created as part of Android programming practice to understand the fundamentals of mobile application development.

## How to Run

1. Clone the repository

3. Build and run the application on an emulator or Android device
